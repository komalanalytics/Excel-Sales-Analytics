# Sales & Financial Analytics Dashboard Project

## Project Overview
This project focuses on analyzing business sales and financial data using Microsoft Excel to generate meaningful insights and support data-driven decision-making. The analysis includes customer performance evaluation, market performance tracking, and profit & loss reporting across different fiscal years and markets.

## Key Reports & Analysis
- Customer Performance Analysis
- Market Performance vs Sales Targets
- Profit & Loss Analysis by Fiscal Year
- Profit & Loss Analysis by Market
- Monthly Financial Performance Reporting

## Tools & Techniques Used
- Microsoft Excel
- Pivot Tables & Pivot Charts
- Power Query
- Conditional Formatting
- Data Cleaning & Transformation
- Financial Reporting & Data Visualization

## Responsibilities & Work Performed
- Collected, cleaned, and organized raw business datasets
- Performed sales and financial data analysis using Excel
- Created interactive reports and dashboards for performance tracking
- Analyzed customer and market trends to identify business insights
- Compared actual sales performance against targets and KPIs
- Generated visual reports to improve data interpretation and reporting efficiency

## Project Outcome
The project provided valuable insights into customer behavior, sales trends, market performance, and company profitability. It improved reporting accuracy and demonstrated how Excel analytics can support strategic business decisions through effective data visualization and reporting techniques.
